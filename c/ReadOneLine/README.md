# ReadOneLine – 파일 한 줄 읽기 함수 테스트

ReadOneLine 프로젝트는 파일 디스크립터로부터 한 줄씩 읽어 반환하는 함수 read_one_line()의 개발을 목표로 합니다.
이 프로젝트의 핵심은 "한 줄 읽기 함수"의 기능 구현에 있으며, 함께 제공된 Makefile은 해당 함수를 쉽게 테스트할 수 있도록 도와줍니다.

## 1. 빌드 및 테스트 방법
### 기본 테스트 (자동 수행)
```bash
make

# 또는 명시적으로
make test
```

**테스트 동작 흐름**
1. `test.txt` 파일 자동 생성
2. `read_one_line_test` 실행 → 각 줄을 출력
3. `test.txt` 와 프로그램 출력 결과 비교
4. 성공/실패 여부를 ANSI 색상 + 아이콘으로 표시 ✅❌

### 클린 및 재빌드
```bash
make clean   # 오브젝트 파일 제거
make fclean  # 실행 파일까지 전체 제거
make re      # 전체 클린 후 재빌드
```

## 2. 참고사항
- 함수 구현은 `read`, `strchr`, `strdup` 등 표준 C 라이브러리를 적극 활용하였습니다.
- 보조 함수는 최소화하고 가독성과 유지보수를 고려한 구조입니다.
- 버퍼 크기는 `BUFFER_SIZE` 매크로를 통해 조절 가능하며, 기본값은 42입니다.
